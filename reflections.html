Skolem’s Paradox

Jack Westbrook
University of Wisconsin–Madison
July 23, 2025

In this essay, I explore Skolem’s paradox—what it is, why it appears paradoxical, how mathematics resolves it, and the philosophical implications surrounding it.

Introduction

In the nineteenth century, mathematicians realized that all of mathematics could be encoded using sets, with the only primitive relation being set membership (\( \in \)). This insight led to Zermelo–Fraenkel set theory with the Axiom of Choice (ZFC), now the foundational system for modern mathematics.

One key question in set theory is when two sets are "the same size." Cantor showed that infinite sets can come in different sizes: for any set \( A \), its power set \( 2^A \) is strictly larger. Yet, the Löwenheim–Skolem theorems tell us that ZFC has countable models—ones whose domain is the same size as the natural numbers \( \mathbb{N} \). This seems contradictory: how can a model contain uncountable sets if its domain is countable?

Mathematical Resolution

The resolution lies in understanding what a model of set theory means. A model \( M \) may contain a set \( x \) it believes is uncountable (i.e., it believes no bijection exists between \( x \) and \( \mathbb{N} \)), yet from the outside, we can construct such bijections because \( M \)'s domain is countable. The paradox is semantic, not mathematical—it arises from interpreting “uncountability” inside vs. outside the model.

This subtlety traces to how quantifiers are interpreted: in the model-theoretic sense, existential quantifiers range over the model’s domain only. Thus, \( M \models \text{“}x \text{ is uncountable”} \) may be true internally, even if externally \( x \) is countable.

Moreover, the paradox relies on first-order logic. Second-order set theory can avoid these issues by quantifying over all subsets directly, and constructivist logics also bypass some of the same problems. Hence, the Löwenheim–Skolem theorems—and the paradox—are not universal but logic-dependent.

Philosophical Reflections

Skolem advocated an algebraic conception of set theory: sets don’t exist independently but are merely elements of whatever model satisfies our axioms. This contrasts with realist or Platonist perspectives, which view mathematical objects (like sets) as existing in an abstract, objective realm.

From the Platonist viewpoint, the existence of countable models that contain “uncountable” sets is no problem, because such models are simply incomplete representations of the true set-theoretic universe. On this view, there really is a unique universe \( V \) in which uncountability has a fixed meaning.

I find this perspective more satisfying. It explains why mathematics is so effective and consistent across disciplines: we are not just playing formal games but uncovering truths about a shared abstract structure. Moreover, relying solely on first-order logic feels unnecessarily limiting when more expressive systems like second-order logic offer ways to capture concepts like uncountability more faithfully.

This essay was originally written for a philosophy course at the University of Wisconsin–Madison.
